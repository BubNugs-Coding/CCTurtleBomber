ComputerCraft TNT Placer

Advanced TNT placement and detonation system for ComputerCraft turtles

View on GitHub

⚠️ Warning

This program places and detonates TNT. Use with caution in-game!

Features

3D Pattern Placement

Places TNT in a cross pattern with multiple layers

Smart Navigation

Automated pathfinding and obstacle clearing

Safety First

Built-in confirmations and safe detonation system

Auto-Return

Automatically returns to starting position

Installation

  1. Create a new file on your turtle named TNTPlacer.lua
  2. Copy the contents of TNTPlacer.lua into the file
  3. Save the file

Requirements

  • ComputerCraft Mining Turtle
  • TNT blocks
  • Redstone block
  • Fuel (coal, charcoal, etc.)

TNT Placement Pattern

T T C T (C = Center, T = TNT) T

Additional layers follow the same pattern vertically if more TNT is specified

Usage

> TNTPlacer
Enter current coordinates (X,Y,Z)
Enter facing direction (0=N,1=E,2=S,3=W)
Enter target coordinates
Enter TNT amount
Type 'CONFIRM' to begin
